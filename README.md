# Propertyware

Propertyware is a cloud-based property management software platform, a RealPage company, offering tools for single-family and low-density rental portfolio management. Its open REST API enables two-way data exchange between Propertyware and third-party systems, supporting custom integrations for properties, units, leases, tenants, work orders, maintenance requests, financial transactions, and owner reports.
